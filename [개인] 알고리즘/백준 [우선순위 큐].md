## 16466. 콘서트

```python
'''
1. 이미 들어 있는거 확인할 때는 "in" 보다 visited (False, True)가 빠르다.
2. 중복된 경우 확인 and 티켓번호가 모두 1일 때 2가 출력 되지 않았음
'''

# 16466. 콘서트
import heapq

N = int(input())

info = list(map(int, input().split()))

visited = [False] * 1000001

q = []

for i in range(len(info)):
    if visited[info[i]] is False:
        heapq.heappush(q, info[i])
        visited[info[i]] = True

idx = 1

check = False

while q:
    ticket_num = heapq.heappop(q)

    if ticket_num == idx:
        idx += 1

    else:
        print(idx)
        check = True
        break

if not check:
    print(idx)

```



## 1417. 국회의원 선거

```python
'''
매 순간(q에서 뽑을 때 마다) 최댓값을 비교한다. => 우선순위 큐
'''
# 1417. 국회의원 선거
import sys, heapq

N = int(input())

dasom = int(input())

q = []

for _ in range(N - 1):
    candi = int(input())

    heapq.heappush(q, -candi)   # 최대 힙

buy = 0

while q:
    candi = -heapq.heappop(q)

    if dasom > candi:
        break

    buy += 1    # 매수
    dasom += 1  # 표 획득
    candi -= 1  # 다른 후보 표 제거

    heapq.heappush(q, -candi)

print(buy)

```
