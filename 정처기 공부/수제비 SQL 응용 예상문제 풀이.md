22. 소프트웨어 생명주기 모델의 종류와 설명

```
[폭프나반]

폭포수 모델 : 각 단계를 확실히 마무리 후에 다음 단계로 진행
프로토타이핑 모델 : 고객이 요구한 기능을 프로토타입으로 만들어 진행
나선형 모델 : 위험을 최소화 하기 위한 모델 (계획 - 위험 분석 - 개발 - 고객 평가)
반복적 모델 : 제품 일부분을 반복적으로 개발하여 최종 시스템으로 완성하는 모델
```
